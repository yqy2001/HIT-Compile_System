1、中间代码的形式

逆波兰表示：

优点：表达式的运算顺序就是运算符出现的顺序，不需要使用括号

| 中缀表示 | 前缀表示 | 后缀表示 |
| -------- | -------- | -------- |
| a+b      | +ab      | ab+      |
| a*(b+c)  | *a+bc    | abc+*    |

三地址码：

这种代码的每条指令最多包含三个地址：两个操作数地址和一个结果地址。
地址的形式可以是：名字（可能是一个指向符号表的指针）、常量、编译器生成的临时变量。

2、声明语句的翻译

声明语句：为程序中用到的变量或常量名指定类型。

类型的作用：

* 类型检查：验证程序是否符合该语言关于类型的相关规则
* 辅助翻译：确定该名字的存储空间。
